## Defining light map

```
"materials": [
    {
        "pbrMetallicRoughness": {
            "baseColorFactor": [ 1.0, 0.0, 0.0, 1.0 ],
            "baseColorTexture": {
                "index": 0
            },
        },
        "extensions": {
            "MOZ_lightmap": {
                index: 1,
                intensity: 1.0
            }
        }
    }
]
```

## Toon material Types

| Property | Type | Description | Requires |
|:------|:------|:------|:------|
| `index` | `integer` | The index of texture. | :white_check_mark: Yes |
| `intensity` | `boolean` | light map intensity | No, default is `1.0` |
